# KizunaSDK (キヅナ) v1.0.0

**Wallet Management SDK for ClawBot**

---

## The Problem

```
❌ User gives ClawBot their wallet private key
   - RISKY: ClawBot has full access to all funds
   - RISKY: If ClawBot is compromised, user loses everything
   
❌ User doesn't want to give wallet to ClawBot
   - But still wants ClawBot to trade NFTs
```

---

## The Solution

```
✅ User funds a wallet created by KizunaSDK
   - User controls the funds
   - ClawBot can only spend what's in the SDK wallet
   - If ClawBot is compromised, user only loses that wallet's funds

✅ KizunaSDK creates and manages wallets
   - ClawBot uses SDK's wallets for trading
   - User can create multiple wallets
   - User can limit what ClawBot can do
```

---

## How It Works

```
┌─────────────────────────────────────────────────────────┐
│                    User                                  │
│  1. Creates wallet via KizunaSDK                        │
│  2. Funds the wallet with crypto                        │
│  3. Gives wallet to ClawBot (via SDK)                  │
└─────────────────────┬───────────────────────────────────┘
                      │
                      ▼
┌─────────────────────────────────────────────────────────┐
│                  KizunaSDK                              │
│  • Creates wallets                                     │
│  • Manages private keys securely                       │
│  • Executes transactions                              │
│  • Tracks portfolio                                   │
└─────────────────────┬───────────────────────────────────┘
                      │
                      ▼
┌─────────────────────────────────────────────────────────┐
│                    ClawBot                              │
│  • Can trade NFTs using SDK wallet                     │
│  • Can check balance                                   │
│  • CANNOT access user's main wallet                    │
│  • CANNOT drain funds to external addresses            │
└─────────────────────────────────────────────────────────┘
```

## Quick Start

```typescript
import { KizunaClient, SUPPORTED_NETWORKS } from 'kizuna-sdk';

// User creates a wallet via SDK
const client = new KizunaClient({
  privateKey: process.env.USER_PRIVATE_KEY, // Or generate new
  rpcUrl: SUPPORTED_NETWORKS[43113].rpcUrl,
  apiKey: process.env.GASLESS_API_KEY,
  chainId: 43113,
});

await client.initialize();

console.log('Trading Wallet:', client.walletService.getAddress());
// User sends funds to this address
```

## Wallet Management

### Create New Wallet (Generated by SDK)
```typescript
// SDK generates a new wallet
const newWallet = client.multiWalletService.createWallet({
  rpcUrl: SUPPORTED_NETWORKS[43113].rpcUrl,
  apiKey: process.env.GASLESS_API_KEY,
  chainId: 43113,
}, 'My Trading Wallet');

await client.multiWalletService.initializeWallet(newWallet.id);

console.log('New wallet address:', newWallet.address);
// User sends crypto to this address
```

### Use Wallet for Trading
```typescript
// ClawBot uses the SDK-managed wallet
const balance = await client.walletService.getBalance();

// Buy NFT
const tx = await client.walletService.transfer(
  '0xNFTSeller',
  '0.5',
  'avax'
);
```

## Security Benefits

| Without SDK | With KizunaSDK |
|-------------|----------------|
| ClawBot has full wallet access | ClawBot only has trading access |
| Risk: Lose all funds | Risk: Only lose trading wallet funds |
| One wallet for everything | Multiple isolated wallets |
| Can't limit ClawBot spending | Can set spending limits |

## Example: User Sets Up ClawBot Trading

```typescript
// 1. User creates a trading wallet via SDK
const wallet = client.multiWalletService.createWallet(config, 'Trading Wallet');

// 2. User sends some AVAX to this wallet address
console.log('Send funds to:', wallet.address);

// 3. ClawBot now trades using this wallet
//    User's main wallet is NEVER exposed to ClawBot

// 4. ClawBot can only trade, not withdraw to external wallets
const tx = await client.walletService.transfer(seller, price, 'avax');
```

---

## Multi-Wallet Example

```typescript
// User creates separate wallets for different purposes
const tradingWallet = client.multiWalletService.createWallet(config, 'Trading');
const savingsWallet = client.multiWalletService.createWallet(config, 'Savings');

// ClawBot only gets access to trading wallet
client.multiWalletService.setActiveWallet(tradingWallet.id);

// ClawBot trades with trading wallet only
// User's savings wallet is completely separate
```
